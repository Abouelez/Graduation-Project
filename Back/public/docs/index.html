Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

Managing Auth

Register User

Add new user to system

Example request:
curl --request POST \
    "http://localhost/api/auth/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"cvdeqnapxiawusveobqa\",
    \"email\": \"tyshawn.ullrich@example.net\",
    \"password\": \"\'h(&\'P`Kl[XqoY&\\/1#wF\",
    \"password_confirmation\": \"repellendus\"
}"

const url = new URL(
    "http://localhost/api/auth/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "cvdeqnapxiawusveobqa",
    "email": "tyshawn.ullrich@example.net",
    "password": "'h(&'P`Kl[XqoY&\/1#wF",
    "password_confirmation": "repellendus"
};

    method: "POST",
    headers,
    body: JSON.stringify(body),

Filters

Example request:
curl --request GET \
    --get "http://localhost/api/courses/filter?keyword=HTML&min=20&max=1000&order=asc&limit=10" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/courses/filter"
);

const params = {
    "keyword": "HTML",
    "min": "20",
    "max": "1000",
    "order": "asc",
    "limit": "10",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
